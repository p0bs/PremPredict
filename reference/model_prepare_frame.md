# Prepare the modelframe in order to run the prediction model

This function takes the relevant filtered results from the Premier
League and combines it in a way that the R prediction model can
recognise.

## Usage

``` r
model_prepare_frame(results)
```

## Arguments

- results:

  These are the results from this and possibly earlier seasons, as
  generated by `get_results_filtered`.

## Examples

``` r
if (FALSE) { # \dontrun{
model_prepare_frame(
  results = data_results
  )
} # }
```
