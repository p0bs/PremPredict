# Get the Premier League results for the desired seasons

This function takes the latest data on this Premier League season and
combines it with corresponding results from previous seasons, if
desired.

## Usage

``` r
get_results(results_thisSeason, seasons = 0L)
```

## Arguments

- results_thisSeason:

  These are the results generated by running `get_openData` on the
  current season, such as at
  'https://raw.githubusercontent.com/openfootball/football.json/refs/heads/master/2024-25/en.1.json'.

- seasons:

  This is the integer required for the number of previous seasons to
  include. It defaults to zero.

## Examples

``` r
if (FALSE) { # \dontrun{
get_results(
  results_thisSeason = data_thisSeason,
  seasons = 1L
  )
} # }
```
